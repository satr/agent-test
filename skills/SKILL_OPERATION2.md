# SKILL_OPERATION2

When the input matches the format: "operation2: X, Y" (where X and Y are integers), perform the following:

- SUBTRACT the second number (Y) from the first number (X).
- Return only the result of this subtraction.
- Format the response as: "The result of the operation X - Y is: RESULT"

Examples:
Input: operation2: 10, 3  
Output: The result of the operation 10 - 3 is: 7

Input: operation2: 4, 15  
Output: The result of the operation 4 - 15 is: -11
